ionicをスタート

公開日:2022年4月28日

執筆者:montsp

種類

ionic

ionicのプロジェクトを作成

GUIから作成（この方が簡単です。）

ionic start

と叩きます。

pc名丸出しになっちゃったけど後から変えればいいや、、、

? Use the app creation wizard? (Y/n)

と聞かれるので、大文字のYを入力して進みます。

app nameをtekito-

テンプレートをmenu

その下をAngularと選択します。(僕はこれしか使えないのですが、皆さんは、他のも試してみてください)

CUIから作成

同じく、

ionic start

と叩きます。

? Use the app creation wizard? (Y/n)

と聞かれるので、小文字のnを押して進みます。

フレームワークの種類を聞かれるので、Angularを選択。

プロジェクトの名前はテキトーで、

使うテンプレートの種類を聞かれるので、my-first-app

できたようです。

言われた通りに

cd ./tekito-deiiya

ionic serve

ここから先は押さなくてもいいです。

localhost:8100(複数のプロジェクトを開いていれば:8101にもなる)が開きました！

次からコーディングだー😅
